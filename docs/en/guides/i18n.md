# Guide
> This section explains how to manage translations for internationalization of menu items.

The web UI now lives in the dedicated
[apache/skywalking-horizon-ui](https://github.com/apache/skywalking-horizon-ui)
sub-project, which has its own release cadence. Internationalization
translations — including menu names and descriptions — are maintained
there. See the
[Horizon UI contributing guide](https://github.com/apache/skywalking-horizon-ui#contributing)
for the up-to-date instructions on adding or editing translations.
